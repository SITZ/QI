Problem
-------
Write a program which accepts amount as integer and display total number of Notes of Rs. 500, 100, 50, 20, 10, 5 and 1.
Print the output in the exact format (including whitespaces) as shown in example below.

Hint
----
Operators

Example
-------

Input:
575

Output:
Rs.500 : 1
Rs.100 : 0
Rs. 50 : 1
Rs. 20 : 1
Rs. 10 : 0
Rs.  5 : 1
Re.  1 : 0
